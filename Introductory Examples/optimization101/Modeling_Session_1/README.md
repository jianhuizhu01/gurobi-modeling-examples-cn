# 建模课程 1 - 部件分销优化

## 问题
根据假设机器学习模型产生的需求数据，以及运输成本和生产能力，确定从一组生产设施到分销地点的最佳部件生产和运输方案，以最小化成本。

本建模示例还附带一个练习。它包含一些一般性问题和建模问题。
