# 三维井字棋

该模型解决了在三维井字棋棋盘上排列 X 和 O，以最小化完成线条或对角线数量的问题。它展示了如何使用混合整数规划来捕捉简单的逻辑约束。

这是 H. Paul Williams 所著《数学规划中的模型构建》第五版第 272 页和 327-328 页中的示例 17。

这个建模示例属于入门级别。我们假设您对 Python 和 Gurobi Python API 有一定的了解，但您可以从示例中学习到任何缺失的概念。
