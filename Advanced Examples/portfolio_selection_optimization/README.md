# 投资组合选择优化

这个模型是经典的 Markowitz 投资组合选择优化模型的一个示例：通过分配投资以最小化风险。这最适合用矩阵形式来表达，
因此我们使用 Gurobi Python 矩阵接口。基本模型相对简单，我们还通过参数化求解来找到有效边界。
