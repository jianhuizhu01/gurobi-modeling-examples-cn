# 人力资源调度问题

这个Jupyter笔记本描述了一个在服务行业中常见的人力资源调度优化问题。
该问题被表述为一个多目标混合整数规划（MIP）模型。
该模型使用Gurobi Python API实现，并使用Gurobi优化器求解。

并且您具有构建数学优化模型的高级知识。通常，这些示例的目标函数
和/或约束条件比较复杂，或者需要Gurobi Python API的高级功能。
